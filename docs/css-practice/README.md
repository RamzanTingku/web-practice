# Comprehensive CSS Learning Reference

A complete vanilla CSS learning resource covering fundamental to advanced concepts with 20+ interactive UI components and demonstrations. Perfect for mastering CSS from the ground up!

## üéØ Project Overview

This project is designed as a comprehensive learning reference for CSS, covering everything from basic box model concepts to advanced animations and modern layout techniques. All demonstrations are built with **pure CSS** - no JavaScript required!

## üìö Complete Coverage

### 1. Fundamentals
- **Box Model**: margin, padding, border, box-sizing, margin collapse
- **Display Property**: block, inline, inline-block, none, visibility, table
- **Position Property**: static, relative, absolute, fixed, sticky, z-index
- **CSS Variables**: Custom properties for theming and consistency

### 2. Modern Layouts
- **Flexbox**: Complete coverage of flex-direction, justify-content, align-items, flex-wrap, flex-grow, flex-shrink, flex-basis, gap
- **CSS Grid**: Grid templates, auto-fit, minmax(), grid-template-areas, span, place-items, responsive grids

### 3. UI Components (20+)
- **Buttons**: Primary, secondary, outline, sizes, loading states
- **Cards**: Standard, horizontal, image cards, hover effects
- **Forms**: Input fields, selects, checkboxes, radio buttons, validation states
- **Tables**: Striped, hover, bordered, responsive tables
- **Badges & Tags**: Status indicators, pill badges, color variants
- **Alerts**: Success, warning, error, info messages
- **Progress Bars**: Standard, striped, animated, circular
- **Tooltips**: Pure CSS tooltips using pseudo-elements
- **Modal**: Full-screen overlay using `:target` selector
- **Tabs**: Horizontal navigation with active states
- **Accordion**: Expandable sections using checkbox hack
- **Dropdown**: Menu with `:focus-within` pseudo-class
- **Avatars**: Sizes, groups, status badges
- **Breadcrumb**: Navigation trail
- **Pagination**: Page navigation controls
- **Skeleton Loaders**: Shimmer loading effects
- **Spinners**: Multiple loader animations

### 4. Responsive Design
- **Media Queries**: Breakpoints for mobile, tablet, desktop
- **Container Queries**: Responsive components based on container size
- **Responsive Units**: rem, em, vw, vh, clamp()
- **Fluid Typography**: Using clamp() for scalable text
- **Mobile-First Approach**: Optimized for all screen sizes

### 5. Transforms & Animations
- **2D Transforms**: translate, rotate, scale, skew
- **Transitions**: ease, ease-in-out, cubic-bezier timing functions
- **Keyframe Animations**: bounce, pulse, shake, spin, float
- **Hover Effects**: lift, glow, fill, transform
- **Loading Animations**: spinners, dots, pulse effects

### 6. Advanced CSS
- **Pseudo-elements**: ::before, ::after for decorative elements
- **Pseudo-classes**: :first-child, :last-child, :nth-child(), :hover, :focus
- **Filters**: blur, grayscale, brightness, hue-rotate
- **Gradients**: linear-gradient, radial-gradient, conic-gradient
- **Clip-path**: circle, polygon, ellipse shapes
- **Blend Modes**: mix-blend-mode for visual effects
- **Aspect Ratio**: Modern aspect-ratio property
