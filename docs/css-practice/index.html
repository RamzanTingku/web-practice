01

Box Model

Every element is a box: content + padding + border + margin

Visual Box Model

margin
border
padding
content 
.element {
  margin: 20px;      /* Outside space */
  border: 5px solid; /* Border line */
  padding: 15px;     /* Inside space */
  width: 200px;      /* Content width */
}

box-sizing Property

content-box width: 200px + padding + border
border-box width: 200px (includes padding & border) 
/* Default - adds to width */
.box { box-sizing: content-box; }

/* Recommended - includes padding */
* { box-sizing: border-box; }

Margin Collapse

margin-bottom: 30px
margin-top: 20px

‚Üë Gap is 30px (larger wins), not 50px!

.box1 { margin-bottom: 30px; }
.box2 { margin-top: 20px; }
/* Gap = 30px (larger wins)
   Vertical margins collapse! */

Width & Height Units

200px (fixed)
50% (of parent)
80vw (of viewport)
auto (content width)
.fixed    { width: 200px; }
.relative { width: 50%; }
.viewport { width: 80vw; }
.auto     { width: auto; }

Controls how an element generates boxes

block vs inline

Block 1
Block 2 (takes full width)
Inline 1 Inline 2 (flows with text) 
/* Block: full width, new line */
div { display: block; }

/* Inline: flows with text */
span { display: inline; }

inline-block

Box 1
Box 2
Box 3

Best of both: inline flow + block sizing

.box {
  display: inline-block;
  width: 100px;  /* works! */
  height: 80px;  /* works! */
}

none vs visibility: hidden

Visible
Hidden (space kept)
None (removed)
Visible
/* Removes from layout */
.gone { display: none; }

/* Invisible but keeps space */
.invisible { visibility: hidden; }

Cell 3
Cell 4

03

Position Property

Controls how elements are positioned in the document

static (default)

Static Box 1
Static Box 2

Normal document flow. top/left ignored.

/* Default - normal flow */
.element {
  position: static;
  /* top, left, etc. ignored */
}

relative

Normal Box
Relative (top: 10px, left: 20px)
Normal Box

Offset from normal position. Space preserved.

.element {
  position: relative;
  top: 10px;   /* offset down */
  left: 20px;  /* offset right */
}

absolute

Normal Box
Absolute (top: 10px, right: 10px)
Normal Box (no gap!)

Removed from flow. Relative to positioned ancestor.

.parent { position: relative; }
.child {
  position: absolute;
  top: 10px;
  right: 10px;
}

fixed

Fixed elements stay in place during scroll.

Example: The navbar at the top is position: sticky

See the "Back to Top" button at bottom-right ‚Üí

.fixed-btn {
  position: fixed;
  bottom: 20px;
  right: 20px;
  /* Stays on screen! */
}

CSS Grid

Two-dimensional layout for rows AND columns

Basic Grid

1
2
3

grid-template-columns: repeat(3, 1fr) 
.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
}

auto-fit & minmax()

1
2
3
4
5
grid-template-columns: repeat(auto-fit, minmax(120px, 1fr))

Resize window to see responsive behavior!

.responsive-grid {
  display: grid;
  grid-template-columns: 
    repeat(auto-fit, minmax(120px, 1fr));
  /* auto-fill: keeps empty columns */
  /* auto-fit: collapses empty columns */
}

Grid Gap

1
2
3
4
row-gap: 1rem; column-gap: 2rem; 
.grid {
  gap: 1rem;        /* both */
  row-gap: 1rem;    /* rows only */
  column-gap: 2rem; /* columns only */
}

Spanning Cells

span 2 cols
2
span 2 rows
4
5
6
.span-2-cols {
  grid-column: span 2;
}
.span-2-rows {
  grid-row: span 2;
}
/* Or explicit: grid-column: 1 / 3; */

Grid Template Areas

header
sidebar
main
footer
.container {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
}
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }

Grid Alignment

Center
place-items: center (shorthand) 
.grid {
  place-items: center; /* shorthand */
  /* = justify-items + align-items */
  place-content: center;
  /* = justify-content + align-content */
}

Accordion (Pure CSS)

                

                    

                        

                            
                            
                            

                                
CSS (Cascading Style Sheets) is a stylesheet language used to describe the
                                    presentation of a document written in HTML. It controls layout, colors, fonts, and
                                    more.

                            

                        

                        

                            
                            
                            

                                
The CSS box model describes how elements are rendered as rectangular boxes. Each box
                                    has content, padding, border, and margin areas.

                            

                        

                        

                            
                            
                            

                                
Flexbox is one-dimensional (row OR column), while Grid is two-dimensional (rows AND
                                    columns). Use Flexbox for components, Grid for page layouts.
